Definicion con palabras que inicie en D

Tabla

Palabra con D Definicion en Español Definicion en Ingles Eje. Imagen
Data Warehouse / Almacén de Datos Un sistema de base de datos relacional centralizado y altamente estructurado, diseñado específicamente para el análisis de consultas, procesamiento de datos históricos y reportes de inteligencia de negocio. A centralized, highly structured relational database system designed specifically for query analysis, historical data processing, and business intelligence reporting.
Data Lake / Lago de Datos Un repositorio de almacenamiento masivo que guarda enormes cantidades de datos brutos en su formato nativo (estructurados, semiestructurados y no estructurados) hasta que se necesitan para su procesamiento o análisis. A massive storage repository that holds vast amounts of raw data in its native format (structured, semi-structured, and unstructured) until it is needed for processing or analysis.
Denormalization / Desnormalización Una técnica de optimización de bases de datos donde se añade información redundante de forma intencional a un esquema relacional normalizado para eliminar uniones (joins) complejas de tablas y acelerar el rendimiento de la lectura. A database optimization technique where redundant data is intentionally added to a normalized relational database schema to eliminate complex table joins and speed up read performance.
Database Schema / Esquema de Base de Datos La estructura formal que representa la configuración lógica y el diseño de toda la base de datos, incluyendo tablas, campos, tipos de datos y relaciones. The formal skeleton structure that represents the logical configuration and design of the entire database, including tables, fields, types, and relationships.
Deadlock / Interbloqueo (Bloqueo Mutuo) Una situación de concurrencia donde dos o más transacciones de la base de datos quedan bloqueadas indefinidamente, cada una esperando un recurso de datos que está bloqueado por la otra transacción. A concurrency situation where two or more database transactions are blocked indefinitely, each waiting for a data resource locked by the other transaction.
