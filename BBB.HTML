Base de Datos NoSQL /NoSQL Database Bases de datos que no utilizan el modelo relacional tradicional de tablas. Son ideales para manejar datos no estructurados y grandes volúmenes de información a gran escala. Databases that do not use the traditional relational table model. They are ideal for handling unstructured data (like JSON documents or graphs) and massive amounts of data at scale.
Base de Datos Relacional/Relational Database - RDBMS El modelo clásico basado en tablas interconectadas mediante llaves. Utilizan el lenguaje SQL y son perfectas cuando la precisión y las relaciones estrictas de los datos son la prioridad. The classic model based on tables interconnected through keys (primary and foreign). They use SQL and are ideal when data accuracy and strict relationships are the priority.
Broker de Mensajes/Message Broker Un intermediario que recibe mensajes de un servicio y los entrega a otro de forma asíncrona, evitando que los sistemas tengan que esperarse entre sí. An intermediary component that receives messages from one service and delivers them to another asynchronously, preventing systems from having to wait for each other to finish.
Big Data / Datos Masivos Conjuntos de datos extremadamente grandes y complejos que no pueden procesarse o analizarse fácilmente con bases de datos relacionales tradicionales debido a su volumen, velocidad y variedad. Extremely large and complex datasets that cannot be easily processed or analyzed using traditional relational database systems due to their volume, velocity, and variety.
Bottleneck / Cuello de Botella Un punto de congestión en la arquitectura de software o en una tubería de datos que restringe el flujo general de la información, reduciendo el rendimiento de todo el sistema. A point of congestion in a software architecture or data pipeline that restricts the overall flow of data, reducing the performance and throughput of the entire system.
