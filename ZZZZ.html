Zero-Day Vulnerability Un fallo de seguridad en el software que es desconocido para los desarrolladores o fabricantes. Se llama de "día cero" porque los creadores del sistema tienen cero días para solucionar el problema desde que se descubre A software security flaw that is unknown to the developers or vendors. It is called "zero-day" because the system creators have zero days to fix the problem once it's discovered,
Zip Un formato de almacenamiento y compresión de archivos muy utilizado en el desarrollo para agrupar múltiples elementos de un proyecto en un solo archivo comprimido A widely used file archive and compression format in development used to bundle multiple project elements into a single compressed
Z-Index Una propiedad de diseño web en CSS que especifica el orden o nivel de apilamiento vertical de los elementos en una página. A web design property in CSS that specifies the vertical stacking order of elements on a page.
Zero-Trust Architecture Un enfoque y modelo de seguridad informática en desarrollo y sistemas que se basa en la premisa de "nunca confiar, siempre verificar" A cybersecurity framework and approach in development and systems based on the premise of "never trust, always verify." It requires every user and device to go through strict authentication and validation
Zombie Process En sistemas operativos y desarrollo backend, es un proceso que ha completado su ejecución pero que aún permanece en la tabla de procesos del sistema debido a que su proceso "padre" no ha leído su estado de finalización. In operating systems and backend development, a process that has completed execution but still remains in the system's process table because its "parent" process has not yet read its exit status.
