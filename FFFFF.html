Palabra con F Definicion en Español Definicion en Ingles Eje. Imagen
Filtro Un componente o función que toma un conjunto de datos o peticiones y deja pasar solo aquellos que cumplen con ciertas condiciones, descartando el resto. A component or function that processes a set of data or requests, allowing only those that meet specific criteria to pass through while discarding the rest.
Flujo de datos El camino o la ruta que siguen los datos moviéndose desde su origen, pasando por diferentes componentes del software, hasta llegar a su destino final. The path or route that data takes as it moves from its source, through various software components, until it reaches its final destination.
Fragmentación En bases de datos, es el proceso de dividir una tabla grande en partes más pequeñas (fragmentos) para guardarlas en diferentes discos o servidores y mejorar la velocidad. In databases, the process of dividing a large table into smaller parts (fragments) to store them across different disks or servers to improve performance.
Fachada (Patrón) Un patrón de diseño de software que proporciona una interfaz única y simple frente a un sistema complejo de código. Oculta la complejidad interna detrás de una "cara" amigable. A software design pattern that provides a simplified, single interface to a complex body of code, hiding the internal complexity behind a friendly "face".
Fila En bases de datos relacionales, es la línea horizontal de una tabla que contiene un registro único con todos sus datos (por ejemplo, los datos completos de un solo cliente). In relational databases, the horizontal line in a table that represents a single, complete record (for example, all the data belonging to one specific customer).
