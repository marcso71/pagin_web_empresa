Definición con palabras que inicien en Q

Tabla

Palabra con Q Definición en Español Definición en Inglés Eje. Imagen
Quality Attributes (Atributos de calidad) Requerimientos no funcionales (como seguridad, escalabilidad y modificabilidad) que dan forma a la arquitectura del sistema. Non-functional requirements (such as security, scalability, and modifiability) that shape the system's architecture.
Queue-Based Load Leveling Patrón de arquitectura que utiliza una cola para amortiguar tareas pesadas, suavizando las cargas masivas intermitentes. An architectural pattern that uses a queue to buffer heavy tasks, smoothing out intermittent heavy loads.
Quorum (Quórum) El número mínimo de votos o nodos requeridos en un sistema distribuido para realizar de forma segura una transacción u operación. The minimum number of votes or nodes required in a distributed system to safely perform a transaction or operation.
Query Responsibility En arquitecturas de datos como CQRS, el lado dedicado del sistema que se encarga exclusivamente de la lectura y presentación de datos. In data architectures like CQRS, the dedicated side of the system responsible exclusively for reading and presenting data.
Quality Gate (Puerta de calidad) Un hito estructural en la línea de despliegue de la arquitectura que impone estándares de código y diseño antes de avanzar. A structural milestone in the architecture pipeline that enforces code and design standards before deployment.
