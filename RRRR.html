Definición con palabras que inicien en R

Tabla

Palabra con R Definición en Español Definición en Inglés Eje. Imagen
Replicación (Replication) La práctica arquitectónica de compartir información en múltiples nodos para garantizar la redundancia de datos y la alta disponibilidad. The architectural practice of sharing information across multiple nodes to ensure data redundancy and high availability.
Redundancia (Redundancy) La inclusión de componentes adicionales o duplicados en la arquitectura para asegurar la operación continua en caso de fallos. The inclusion of extra or duplicate components in the system architecture to ensure continuous operation in case of failures.
Rate Limiting (Limitación de tasa) Patrón de seguridad arquitectónico que controla el flujo de tráfico enviado o recibido por una interfaz de red o servicio. An architectural safety pattern that controls the rate of traffic sent or received by a network interface or service.
REST (Representational State Transfer) Estilo de arquitectura que define un conjunto de restricciones utilizadas para la creación de servicios web distribuidos. An architectural style that defines a set of constraints to be used for creating distributed web services.
Resilience (Resiliencia) La capacidad de una arquitectura de software para recuperarse adecuadamente de fallos de infraestructura o de red. The ability of a software architecture to recover gracefully from infrastructure or network failures.
