Definicion con palabras que inicie en H

Tabla

Palabra con H Definicion en Español Definicion en Ingles Eje. Imagen
Huella de memoria La cantidad de memoria RAM que un software o una base de datos ocupa mientras se está ejecutando. Si el software es pesado, deja una "huella" grande en el sistema. The amount of RAM memory that a software application or database occupies while it is running. A heavy application leaves a large "footprint" on the system.
Hijo (Nodo) En estructuras de datos en forma de árbol (como los índices de una base de datos), es un nodo que depende directamente de otro nodo superior (llamado padre). In tree-like data structures (such as database indexes), a node that extends directly from a higher node (called the parent node).
Hilos (De ejecución) Las secuencias de instrucciones más pequeñas que un software puede ejecutar en paralelo. Permiten que el sistema haga varias tareas al mismo tiempo (como procesar múltiples consultas a la vez). The smallest sequences of programmed instructions that a software can execute in parallel, allowing the system to perform multiple tasks at the same time.
Híbrido (Almacenamiento) Un modelo de arquitectura donde se combinan dos tecnologías diferentes. Por ejemplo, guardar los datos más importantes en servidores propios del edificio y los menos importantes en la nube. An architectural model that combines two different technologies. For example, storing critical data on local on-premise servers and less critical data in the cloud.
Historial de cambios Un archivo o sección de la base de datos donde se registran cronológicamente todas las modificaciones, movimientos o errores que han ocurrido en el sistema. A file or database section where all modifications, movements, or errors that have occurred in the system are recorded chronologically.
