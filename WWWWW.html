Webpack Es un empaquetador de módulos (bundler) de código abierto para aplicaciones JavaScript modernas. Toma todos los elementos de un proyecto (archivos JS, CSS, imágenes) An open-source module bundler for modern JavaScript applications. It takes all project elements (JS, CSS, images files)
Wireframe Es un boceto, esquema o guía visual de baja fidelidad que representa la estructura básica y el diseño de una interfaz de usuario (pantalla o página web). A low-fidelity sketch, blueprint, or visual guide that represents the basic structure and layout of a user interface (screen or webpage).
Workflow Secuencia de pasos, tareas o procesos automatizados que se deben seguir desde el inicio hasta el final para completar un objetivo específico en el desarrollo de software A sequence of steps, tasks, or automated processes that must be followed from start to finish to complete a specific goal in software development
Waterfall Model Un enfoque clásico y lineal para el ciclo de vida del desarrollo de software, donde el proyecto se divide en fases secuenciales de forma estricta (Requisitos, Diseño, Implementación, Verificación, Mantenimiento). A classic, linear approach to the software development lifecycle, where the project is strictly divided into sequential phases (Requirements, Design, Implementation, Verification, Maintenance).
Widget Un componente visual o elemento de la interfaz de usuario reutilizable (como un botón, un cuadro de texto, un calendario o un menú de navegación) A reusable visual component or user interface element (such as a button, a text box, a calendar, or a navigation menu)
