A

Términos que comienzan con la letra A.

# Término (Inglés) Término (Español) Definición Esquema / Representación
01
API GatewayArquitectura
Pasarela de API
 Punto de entrada único que recibe solicitudes de clientes y las enruta de forma inteligente al microservicio correcto. Maneja autenticación y balanceo de carga.
🔀 Cliente ➔ Gateway ➔ Apps
02
ACIDBases de Datos
ACID (Atomicidad, Consistencia, Aislamiento, Durabilidad)
 Conjunto de propiedades que garantizan que las transacciones de una base de datos se procesen de forma segura y confiable sin corromper los datos.
🔒 A · C · I · D = 100% Seguro
03
AggregationBases de Datos
Agregación
 Proceso en el cual se recopilan múltiples datos dispersos para presentarlos en forma de resumen estadístico (ej: operaciones SUM, AVG, COUNT en SQL).
📊 Datos ➔ [∑ Agregación] ➔ Totales
04
AsynchronousArquitectura
Asíncrono
 Modelo de comunicación donde el emisor envía un mensaje u operación y continúa con sus tareas sin esperar una respuesta inmediata del receptor.
⏳ Envío (...) ➔ Procesado posterior
05
AvailabilityArquitectura
Disponibilidad
 Métrica de arquitectura que mide la proporción de tiempo que un sistema permanece operativo, accesible y listo para atender peticiones de los usuarios.
📈 Uptime 99.99% Online
06
AttributeBases de Datos
Atributo
 Cada una de las propiedades, características o campos numéricos/alfanuméricos que definen o componen a una Entidad en un modelo relacional.
📋 Tabla ➔ [Columna / Atributo]
07
AuthenticationArquitectura
Autenticación
 Mecanismo de seguridad que se encarga de validar rigurosamente la identidad de un usuario o servicio antes de permitirle el acceso al ecosistema.
🔑 User + Pass ➔ Token Válido
