Kafka Una plataforma distribuida de transmisión de datos (streaming) de código abierto. En la arquitectura de software, se utiliza para construir canalizaciones de datos en tiempo real y aplicaciones que se adaptan al flujo de datos, sirviendo como un intermediario de mensajes de alto rendimiento y tolerancia a fallos. An open-source distributed event streaming platform. In software architecture, it is used for building real-time data pipelines and streaming apps, serving as a high-throughput, fault-tolerant message broker.
Kubernetes Una plataforma de código abierto para la automatización del despliegue, el escalado y la gestión de aplicaciones en contenedores. An open-source container-orchestration system for automating computer application deployment, scaling, and management.
Kernel El núcleo o componente central de un sistema operativo que actúa como puente entre el software de la aplicación y el hardware. The core program of an operating system that manages operations of the computer and the hardware.
Key-Value Store Un tipo de base de datos no relacional (NoSQL) que utiliza un método simple de clave-valor para almacenar datos. A type of non-relational (NoSQL) database that uses a simple key-value method to store data.
Keep Un método o señal que se envía entre dispositivos para mantener activa una conexión de red (como HTTP o TCP) y evitar que se cierre por inactividad. A message or signal sent between devices to maintain a network connection (like HTTP or TCP) active, preventing it from breaking due to inactivity.
