ACID (Atomicity, Consistency, Isolation, Durability) / Atomicidad, Consistencia, Aislamiento, Durabilidad Un conjunto de propiedades que garantizan que las transacciones en una base de datos sean 100% confiables y no corrompa la información si algo falla. A set of properties that guarantee database transactions are processed reliably. If an error occurs, the system rolls back to prevent data corruption.
Aggregation / Agregación El proceso de recopilar, organizar y resumir datos detallados de múltiples registros de la base de datos para realizar análisis estadísticos o cálculos de alto nivel (ej. sumas, promedios, conteos). The process of gathering, organizing, and summarizing granular data from multiple database records to perform statistical analysis or high-level calculations (e.g., sums, averages, counts).
Attribute / Atributo Un componente de la base de datos que representa una característica o propiedad específica de una entidad, lo cual corresponde a una columna en una tabla relacional. A database component that represents a specific characteristic or property of an entity, which corresponds to a column in a relational table.
Aggregate Function / Función de Agregación Una función integrada de la base de datos (como SUM, AVG, COUNT) que toma múltiples valores de una columna y devuelve un único valor resumido. A built-in database function (like SUM, AVG, COUNT, MAX, MIN) that takes multiple values from a column and returns a single summary value.
Anomaly (Database Anomaly) / Anomalía de Base de Datos Una inconsistency o error que ocurre en bases de datos no normalizadas durante la modificación de datos, categorizada como anomalía de inserción, actualización o borrado. An inconsistency or error that occurs in unnormalized databases during data modification, categorized as insertion, update, or deletion anomalies.
