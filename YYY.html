Yarn Es un gestor de paquetes rápido, seguro y confiable para el entorno de JavaScript. Fue desarrollado por Facebook (Meta) y se utiliza para automatizar la instalación, actualización, configuración y eliminación de dependencias y librerías en un proyecto. Developed by Facebook (Meta), it is used to automate the installation, update, configuration, and removal of project dependencies and libraries.

Yield Una palabra clave utilizada en varios lenguajes de programación (como JavaScript o Python) dentro de funciones llamadas "generadores". Se usa para pausar la ejecución de la función y devolver un valor al código que la llamó A keyword used in several programming languages (such as JavaScript or Python) inside functions called "generators." It is used to pause the function's execution and return a value to the caller
YottaByte Una unidad de almacenamiento de información digital que equivale a $10^{24}$ bytes (o un septillón de bytes). Aunque es un tamaño masivo, se menciona en la arquitectura de Big Data y sistemas en la nube para describir el volumen global de datos en el desarrollo moderno. A unit of digital information storage equal to $10^{24}$ bytes (or one septillion bytes). Although massive in size, it is referenced in Big Data architecture and cloud computing to describe global data volumes in modern development.
